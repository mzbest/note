## 蒜炒油菜

| 食材   | 用量     |
| ------ | -------- |
| 蒜     | 四五瓣   |
| 葱     | 两三颗   |
| 姜     | 一点     |
| 上海青 | 一两人份 |
| 黄酒   | 2勺      |
| 耗油   | 2勺      |
| 盐     | 一些     |
| 味精   | 一些     |

**步骤**

1. 宽水，菜要少，洗干净
2. 将蒜，耗油，黄酒，盐，味精调料汁
3. 菜叶烫一下，别让开锅，绿了就行
4. 葱姜爆香
5. 加入青菜
6. 加入盐
7. 放入怼好的耗油汁，不加水，拌匀就好
8. 色泽翠绿，不干巴（放入一些猪大油更好）
